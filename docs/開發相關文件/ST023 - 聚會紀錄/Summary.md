# ST023 - 聚會紀錄 (Group Meeting Records)

**Priority**: Could Have (Not MVP Scope)  
**Phase**: 8 - 牧養與關懷模組

## 簡述
實作小組聚會紀錄功能，小組長可紀錄每週聚會出席、缺席、請假、新人、奉獻金額、以及聚會主題與氣氛量表。

## 核心功能
- 每週聚會點名
  - 出席、缺席、請假、新人
- 奉獻金額記錄
- 聚會內容紀錄
  - 聚會主題
  - 氣氛量表（1-5 星）
- 歷史聚會記錄查看
- 出席率統計
