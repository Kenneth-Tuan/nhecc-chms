# ST011 - 全域戰情室 (Global Dashboard)

**Priority**: Should Have  
**Phase**: 5 - 儀表板模組

## 簡述
實作系統管理員的全域儀表板，顯示全教會 KPI、成長趨勢、牧區分佈、以及待辦事項警示。

## 核心功能
- 關鍵指標卡片（活躍會友數、主日出席、受洗人數、開課數）
- 成長趨勢圖表（近六個月折線圖）
- 牧區分佈統計（圓餅圖）
- 熱門課程類別
- 待辦事項警示（紅/橘卡片）
- 快速操作區
