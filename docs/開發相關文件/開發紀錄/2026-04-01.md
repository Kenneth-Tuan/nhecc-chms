# 開發紀錄 - 2026-04-01

## 今日開發重點：班級管理系統 (Phase 1)

今日完成了「班級管理」功能的基礎建設，包含後端教師清單 API、班級列表過濾 API，以及前端響應式的開班流程。

### 1. 後端架構優化

- **專屬教師 API (`GET /api/courses/classes/teachers`)**
  - 在 `MemberService` 中封裝了 `listTeachers()`。
  - 邏輯：根據角色權限（具備 `course:manage` 或 `course:grade`）動態過濾出可用於授課的人員名單。
  - 目的：解決以前需抓取全體會友再過濾的效能問題。
- **班級列表 API (`GET /api/courses/classes`)**
  - 建立 `CourseClassService` 處理跨表彙整邏輯。
  - 支援按「教師 ID」、「課程分類（透過 Template 關聯）」與「班級狀態」進行過濾。
  - 回傳資料已自動彙整 `templateName` 與 `templateCode`，減少前端 Join 負擔。

### 2. 前端頁面開發

- **班級管理主頁面 (`/dashboard/courses/classes/index.vue`)**
  - 採用 PrimeVue `DataTable` 實作。
  - 具備響應式設計：在寬度不足時自動轉為堆疊式 (Stack) 佈局，方便手機端檢視。
  - 整合頂部篩選列：可同步按分類、老師、狀態即時更新列表。
- **導引式開班流程 (`/dashboard/courses/classes/create.vue`)**
  - **兩步驟設計**：
    1.  **選擇教案 (Template Selection)**：提供所有啟用的教案清單供管理員挑選。
    2.  **填寫詳情 (Class Details)**：繼承教案資訊，設定具體的老師、日期、地點與人數上限。已優化日期輸入，改用 PrimeVue `DatePicker` 以提升操作體驗與瀏覽器相容性。
  - **適配性**：完全共用程式碼於行動端與桌面端，解決了彈窗 (Dialog) 在手機上難以輸入的問題。

### 3. 資料模型與驗證

- **`CourseClass` 型別更新**：
  - `teacherIds: string[]` 改為 `teachers: { id, name }[]`，達到去正規化顯示。
  - 統一了 `CourseAttachment` 與 `ClassAttachment` 的結構，增加 `updatedAt` 追蹤欄位。
- **Zod 驗證同步**：
  - 更新了 `courseClassSchema` 以確保手動輸入的班級資料符合規格。

### 4. UI 基礎設施與組件標準化 (ST015)

為了確保後台管理系統的一致性與開發效率，抽離並統一了頁面佈局組件：

- **統一頁面容器 (`BasePageContainer`, `BasePageHeader`)**:
  - 標準化了標題列、返回導航、動作按鈕區（Actions Slot）的排版。
  - 內建響應式斷點與內容最大寬度控制，解決了不同螢幕尺寸下排版鬆散的問題。
- **動態對話框系統 (`useAppDialog`)**:
  - 封裝 PrimeVue `DynamicDialog` 工具化的進度。
  - 提供 `showSuccess` 與 `showError` 函式式調用，大幅減少在 `.vue` 文件中手動聲明 `<Dialog>` 的 Boilerplate。
  - 統一了報名成功、錯誤提示的視覺風格（AppMessageDialog）。

### 5. 教案（課程模板）管理優化

針對系統核心的教案管理進行了細節對齊與視覺升級：

- **術語規範化**：全面校對並修改系統文字，嚴格區分「教案 / 課程模板 (Course Template)」與「班級 (Course Class)」，避免管理邏輯混淆。
- **介面視覺升級 (`TemplateForm`)**:
  - **無障礙適配**：字體大小統一以 16px (text-base) 為基礎，強化標籤字重 (font-bold) 與輸入框對比度。
  - **操作流暢化**：教案表格直接整合「複製教案」與「快速開班」入口，實現「教案 -> 班級」的一鍵操作流。

### 6. 探索課程介面重構 (SAT.cool Style)

為了提升大齡使用者的操作體驗，參考 [SAT.cool](https://sat.cool/home) 的極簡設計風格對前台「探索課程」進行了全面重構。

- **列表頁面優化 (`/explore/index.vue`)**
  - **極簡 4 欄網格**：移除雜亂按鈕，改為大標題、高對比度的排版。
  - **分類過濾進度**：重新設計 `SelectButton` 並支援深色模式 (Dark Mode)。
- **課程詳情頁面新上線 (`/explore/[id].vue`)**
  - **清爽 Hero 視角**：大標題展示、課程代號區塊化。
  - **智慧報名側邊欄**：
    - 即時先修條件 (Prerequisite) 檢查與報名資格判定。
    - 名額視覺化進度條。
    - 整合 `useAppDialog` 提供沉浸式的報名成功反饋。
- **課程卡片組件 (`CourseCard.vue`)**
  - 全面簡化，移除直接報名按鈕，改為連結導向詳情頁。
  - 加入細膩的懸浮 (Hover) 升起效果與深色模式適配。
- **後端單一詳情 API (`GET /api/explore/courses/[id]`)**
  - 在 `CourseClassService` 中實作 `getPublishedById`，確保僅發佈中的課程可被公開探索。
